# claude-levelup

How Pathfinders runs coordinated Claude Code sessions through GitHub Copilot review and human gates — spec-driven, on a foundation of engineering craft — to accelerate without sliding into vibe-coding.

- **The argument** — [`index.html`](index.html): *Claude burns the tokens; the team collects the XP.*
- **The playbook** — [`playbook.html`](playbook.html): two worked scenarios, the kit to run it on your team, and the references behind the practice.
- **The method in action** — two random architecture katas (drawn from architecturalkatas.com) run through the lenses and the loop end-to-end:
  - [`kata-doorbell.html`](kata-doorbell.html): a wifi doorbell + camera (reliability, coupling, the edge).
  - [`kata-tickets.html`](kata-tickets.html): high-volume concert ticketing (scale, contention, load-shedding).

Four self-contained static pages (no build, no dependencies). Published via GitHub Pages.
